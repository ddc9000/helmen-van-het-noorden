# Helmen van het Noorden

Een elegante digitale ode aan de iconische helmen uit het Germaanse, Noorse en Angelsaksische cultuurgebied.

Van de vroege Bronstijd (~2000 v.Chr.) tot de Vikingtijd en vroege Middeleeuwen (~1100 n.Chr.) in het uitgebreide gebied van de Noordzee, Oostzee en aangrenzende Germaanse streken.

Focus op de prachtige ontwerpen, de gebruikte materialen (ijzer, brons, goud, granaat, maliën...) en het ambachtelijke meesterschap.

## De spirituele wens

**Moge alle wezens Valhalla of Nirvana bereiken, bevrijd van lijden.**

## Lokaal draaien

```bash
git clone https://github.com/ddc9000/helmen-van-het-noorden.git
cd helmen-van-het-noorden
# Open index.html in je browser
```

## Deployen op GitHub Pages

1. Ga naar de repo op GitHub
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: `main` /root
5. Save

De site is dan binnen enkele minuten live op `https://ddc9000.github.io/helmen-van-het-noorden/`

## Scope & uitbreiding

De pagina is bewust uitbreidbaar. Nieuwe helmen uit het bredere gebied (tot ~1000 km radius) en periode zijn welkom. Zie CONTRIBUTING.md voor hoe je makkelijk een nieuwe kaart toevoegt.
